Library Changes

Important news about the Local Library! They are going to be adding a new system to return books in order to accomodate covid precautions. With this new system there will be larger waits in order to check out books, but you wont have to worry about getting sick from them. Books will now be kept in bins for 2 weeks after being returned to allow time for any traces of the virus to die off of them. books will still be returned into the same place as before, just expect to have to wait longer for a book if it isnt readily available at the library.
